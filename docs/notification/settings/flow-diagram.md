# แผนภาพ Flow การทำงาน (Mermaid)

```mermaid
flowchart TD
  A(เข้าสู่เมนูตั้งค่าการแจ้งเตือน) --> B(โหลดรายการตั้งค่าปัจจุบัน)
  B --> C{เลือกประเภท/ช่องทาง}
  C -->|เปลี่ยนสถานะ| D(เปิด/ปิด is_enabled)
  C -->|แก้ไขช่วงเวลา| E(ปรับ time window)
  C -->|เพิ่มประเภท/ช่องทางใหม่| F(สร้าง record ใหม่)
  D --> G(บันทึก)
  E --> G
  F --> G
  G --> H(บันทึกสำเร็จ)
  H --> B
